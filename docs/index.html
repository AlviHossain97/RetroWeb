PiStation

A two-host retro gaming and media platform powered by Raspberry Pi 3 and a local AI-driven laptop backend.

System Architecture

PiStation runs as two physical hosts cooperating over the LAN so each can do what its hardware is suited for:

📺 Raspberry Pi 3 (at the TV)

Quad-core ARM, 1 GB RAM. Acts as the TV-attached client handling native retro emulation, Kodi media playback, and Moonlight thin-client for the dashboard.

💻 Laptop (RTX 3070)

Handles the analytics backend, MySQL, Jellyfin server, local STT/TTS models, and streams the React dashboard back to the Pi via Sunshine NVENC.

Interaction Loops

  • Loop 1 — Native gaming: EmulationStation launches RetroArch locally on the Pi for low-latency native emulation. Runcommand hooks sync sessions to the laptop's backend.
  • Loop 2 — Media: Kodi on the Pi streams transcoded media from the laptop's Jellyfin server.
  • Loop 3 — Dashboard & AI: The React 19 PWA is streamed to the Pi over Sunshine. It features a fully local conversational voice assistant powered by Kokoro, Whisper, and Ollama.

Homebrew Corpus

To avoid commercial-ROM copyright issues, three GBA homebrew titles were written specifically for this project:

  • Red Racer - Rapid Python prototype to C port.
  • Mythical - Python prototype to C++ port.
  • Bastion Tower Defence - Python+SDL2 prototype to Butano C++ port.
